# DC-Biased-Polarization-Modulation-in-Honeycomb-Serpentine-Interferometer
Honeycomb-Serpentine Polarization Interferometer: 3V DC Bias Effects" "Electrically Modulated Polarization States in DIY Honeycomb Interferometer"
## Experimental Summary

- **Setup**: Honeycomb-Serpentine polarization interferometer with fixed input privacy film and N20 motor rotated output polarizers.
- **Condition**: 3 V DC bias applied via copper wire + aluminum tape electrode.
- **Key Observation**:
  - Significant increase in temporal brightness fluctuation under DC bias (σ: 0.35 → 1.25).
  - Slight enhancement in color saturation.
  - Suggests effective electrically induced polarization state modulation.
